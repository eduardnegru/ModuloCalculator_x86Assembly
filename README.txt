NEGRU ADRIAN EDUARD 322CC

In registrul eax pun deimpartitul(numarul fiind pe 32 de biti deimpartitul este pus in eax), iar in registrul ebx pun impartitorul adica baza.
Impart succesiv la baza, catul devine deimpartit, iar restul il pun pe stiva.
Restul impartirii lui eax la ebx este pus in edx. Catul este actualizat si este pus in eax (catul
devine deimpartit).
Pentru a afisa rezultatul golesc stiva si afisez fiecare element cu PRINT_CHAR(in ordine inversa
a obtinerii resturilor).Pentru a verifica daca stiva este goala compar ebp cu esp.
